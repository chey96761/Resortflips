How the math works.

Each timeshare points system uses its own arbitrary scale — there's no universal "point" the way there's a universal dollar. But each system's points buy approximately the same lodging value per point in their respective networks, and that's how we convert.

Equivalence basis

Our conversion uses median per-point reservation value across each system — what one point gets you in nightly lodging at a mid-tier property in mid-season. The ratios are approximately:

1 MVCP ≈ 50 StarOptions ≈ 23 HGV Points ≈ 70 Wyndham Points

Why it's an approximation

Each system has wildly varying values within itself — a peak-season 2BR Oceanfront StarOption costs roughly 3x what an off-season 1BR costs. The same is true for MVCP, HGV, and Wyndham. So conversions are directional — useful for comparing ownerships, less useful for booking specifics.
